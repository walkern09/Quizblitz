⚙️ One-time Setup

QuizBlitz needs a free Firebase database to sync players in real time. Takes about 2 minutes.

1

Create a free Firebase project

Go to console.firebase.google.com → Add project → name it anything (e.g. "quizblitz") → continue through the steps (disable Analytics if you like).

2

Add a Realtime Database

In your project: Build → Realtime Database → Create database. Choose any region. When asked for rules, select "Start in test mode" (allows open read/write — fine for a game).

3

Paste your database URL below

It looks like: https://your-project-default-rtdb.firebaseio.com
Find it at the top of the Realtime Database page.

Your Firebase URL is saved in your browser. You only need to do this once — it'll be remembered next time you open this file.
